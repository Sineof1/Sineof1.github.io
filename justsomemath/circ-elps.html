Transformations of Circle Parts

The statement "a circle is an ellipse" is a statement.

In "transformational form," \(\small\mathtt{g(x) = \color{green}{A}f(\color{purple}{B}(x - \color{brown}{C}) + \color{red}{D}}\), changing the A-value stretches or shrinks the graph .

We can think about the circle at the left (code) as being created (traced) by the end of the black hypotenuse of the right triangle. That would mean that this hypotenuse segment has a constant length as the radius of the circle, \(\small\mathtt{r}\). If we imagine that this segment starts at the origin \(\small\mathtt{(0, 0)}\) then the end of the segment is given by \(\small\mathtt{\color{red}{x}^2 + \color{blue}{y}^2 = r^2}\).

And lo, this is the formula for a circle, with the more general \(\small\mathtt{(x - h)^2 + (y - k)^2 = r^2}\) introducing \(\small\mathtt{h}\) and \(\small\mathtt{k}\) to represent offsets from the origin.

Solving the equation \(\small\mathtt{x^2 + y^2 = r^2}\) for \(\small\mathtt{y}\) can help reveal two "pieces" of a circle, since the solution equation is a square root; so we have a "plus-or-minus" situation: \(\small\mathtt{y = \pm \sqrt{r^2 - x^2}}\). I've shown an example on the graph below. Here you've got a circle with a radius, \(\small\mathtt{r}\), currently set to 1 (though you can slide to change it). The top piece of the circle is a function whose graph is "above" the x-axis—which represents the positive square root—and the bottom piece, the negative square root, is a mirror-image function of the top piece reflected over the x-axis.

Now Let's Squish It!

Now we want to stretch this graph horizontally and vertically because I told you we do. And we can think about doing this to each of the function pieces above. So, we can use what is maybe the greatest equation in all of K–12dom: \[\small\mathtt{g(x) = \color{green}{A}f(\color{purple}{B}(x - \color{brown}{C}) + \color{red}{D}}\]

This equation appears in different disguises across different lessons, but most would call it "transformational form." Each of the capital letters on the right side of the equation has a special meaning related to transformations of the graph. There are more details to be had, but these are the basic ideas:

  • Changing the value of \(\mathtt{\color{green}{A}}\) affects the vertical stretch of the graph.
  • Changing the value of \(\mathtt{\color{purple}{B}}\) affects the horizontal stretch of the graph.
  • Changing the value of \(\mathtt{\color{brown}{C}}\) affects the horizontal shift of the graph.
  • Changing the value of \(\mathtt{\color{red}{D}}\) affects the vertical shift of the graph.

Our "top piece" function, which is written as \(\small\mathtt{y = \sqrt{r^2 - x^2}}\) in the graph above, we can write, in transformational form as \[\small\mathtt{y = \color{green}{1} \cdot \sqrt{\color{purple}{-1}(x^2 - \color{brown}{0}) + r^2} + \color{red}{0}}\] We start (ignoring signs) with \(\small\mathtt{A = \color{green}{1}}\), \(\small\mathtt{B = \color{purple}{1}}\), \(\small\mathtt{C = \color{brown}{0}}\), and \(\small\mathtt{D = \color{red}{0}}\). Type this function in on the graph above to play with how changing these values changes the graph. The B-value is kind of tricky. Whereas multiplying the A-value by 2 stretches the graph vertically by 2; the B-value is really the denominator of a unit fraction.
